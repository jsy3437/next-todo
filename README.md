##  설치
다음은 앱 설치 및 설정, 실행하는 방법의 예시 입니다.


  1.  저장소 복제
  ```javascript
    git clone https://github.com/jsy3437/next-todo.git
  ```
  2.  npm 이나 yarn로 패키지 설치
  ```javascript
    npm install
    또는
    yarn add
  ```
  4.  .env를 추가
  ```javascript
    .env
    
    NEXT_PUBLIC_API_URL=http://localhost:3000
  ```
